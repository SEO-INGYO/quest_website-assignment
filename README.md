# 퀘스트를 이용한 동기 부여 웹사이트


## 프로젝트 기간
2021/11/1 ~ 2021/12/14

## 사용 기술
<img src="https://img.shields.io/badge/C Sharp-239120?style=flat-square&logo=csharp&logoColor=white"/> <img src="https://img.shields.io/badge/Visual Studio-5C2D91?style=flat-square&logo=Visual Studio&logoColor=white"/> <img src="https://img.shields.io/badge/.NET-512BD4?style=flat-square&logo=.NET&logoColor=white"/> <img src="https://img.shields.io/badge/Microsoft SQL Server-CC2927?style=flat-square&logo=Microsoft SQL Server&logoColor=white"/>

## 내용
웹 프로그래밍 및 데이터베이스 응용 강의를 수강하면서 진행했던 프로젝트입니다. 사용자가 퀘스트를 수락하고 완료하여 레벨을 올리면서 랭킹과 함께 과제 및 일거리에 동기를 부여해 줄 수는 있는 웹사이트입니다.

## 파일 요약
- CreateAccount_MainPage.aspx : 계정 생성 html 코드
- CreateAccount_MainPage.aspx.cs : 계정 생성 c# 코드
- Default.aspx : 기본 html 코드
- Default.aspx.cs : 기본 c# 코드
- Login_MainPage.aspx : 로그인 html 코드

- Manager_MasterPage.master : 관리자의 마스터 페이지 html
- Manager_MasterPage.master.cs : 관리자의 마스터 페이지 c#
- MyPage_UserPage.aspx : 사용자의 정보 관리 html 코드
- MyPage_UserPage.aspx.cs : 사용자의 정보 관리 c# 코드

- Quest_UserPage.aspx.cs : 사용자의 퀘스트 관리 c# 코드
- Ranking_UserPage.aspx : 사용자의 랭킹 html 코드

- SearchAccount_MainPage.aspx : 계정 찾기 html 코드

- User_ManagerPage.aspx : 관리자의 사용자 정보 관리 html 코드
- User_ManagerPage.aspx.cs : 관리자의 사용자 정보 관리 c# 코드
- User_MasterPage.master : 사용자의 마스터 페이지 html 
- User_MasterPage.master.cs : 사용자의 마스터 페이지 c#
- Counter.txt : 접속자 체크 텍스트 파일
- Style.css : 스타일 시트 파일
